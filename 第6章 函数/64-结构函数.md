##结构函数

&emsp;&emsp;第 5 章介绍了结构类型，它可在一个地方存储多个数据元素，但实际上结构可以做的工作远不止这一点。例如，除了数据，结构还可以包含函数。这初看起来很奇怪，但实际上是非常有用的。

例如，考虑以下结构：

```javascript
        struct CustomerName
        {
            public string firstName, lastName;
        }
```
